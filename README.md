# Laravel_Blog
A complete blog develop with Laravel with all basic blog capabilities;
> Backend Login,
> Image Upload,
> WYSIWYG Texteditor,
> Comment,
> Tag,
> Categories.

### NB: This is a blog with the basic features working. You will have to customize the design to suit you.
